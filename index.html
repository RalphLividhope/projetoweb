Seu surgimento

Retornar

Surgiu graças à ideia de Tim Berners-Lee, que trabalhava no Centro Europeu de Pesquisa Nuclear CERN, que divulgou um comunicado à imprensa sobre seu projeto que buscava acessar arquivos on-line que funcionavam em redes de computadores ou máquinas eletrônicas baseadas no protocolo TCP/IP.

“uma plataforma aberta que permitiria a todo mundo e em todos os lugares compartilhar informações, ter oportunidades de acesso e colaborar através de barreiras geográficas e culturais”

No início da década de 1990, o HTML foi finalmente definido como um subconjunto do SGML1.

No conceito

Retornar

HTML, é traduzido literalmente como Hypertext Markup Language (Linguagem de marcação de hipertexto).

Trata-se de uma linguagem de marcação, definida como uma forma de codificar um documento que, junto com o texto, incorpora tags ou marcações que contêm informações adicionais sobre a estrutura do texto ou sua apresentação. A presença dessas linguagens é encontrada até mesmo no setor editorial.

HTML é um padrão que serve de referência para o software ligado ao desenvolvimento de páginas da Web em suas diferentes versões, define uma estrutura básica e um código (chamado código HTML) para a definição do conteúdo de uma página da Web, como texto, imagens, vídeos, jogos, entre outros. É um padrão a cargo do World Wide Web Consortium (W3C) ou WWW Consortium, uma organização dedicada à padronização de quase todas as tecnologias ligadas à Web, especialmente no que diz respeito à sua escrita e interpretação. O HTML é considerado a linguagem mais importante da Web e sua invenção foi crucial para o surgimento, o desenvolvimento e a expansão da WWW.

Funcionalidade e seu uso na prática

Retornar

HTML (como mencionado acima), é uma linguagem de marcação destinada ao desenvolvimento de páginas da Web, com tags2 como "p"; "h1"; "h3"; "br", que têm a função de formatar o texto da página. As tags mencionadas acima são delimitadas pelos caracteres "<>" e, por fim, encerram sua função delimitadas pelo caracter "/" que é adicionado no meio dos anteriores.

Ele também tem atributos que são adicionados a essas tags, que servem a funções específicas, como endereçar um hiperlink ou simplesmente ajustar o tamanho das letras.

Significados e esclarecimentos

Retornar
  1. SGML: é uma linguagem de marcação de padrão internacional, não proprietário e de código aberto, utilizado para troca eletrônica de dados para ser usada por diferentes sistemas informatizados (portabilidade), garantindo que documentos codificados de acordo com suas regras possam ser transportados de um ambiente de hardware e software para outro, sem perda de informação.
  2. Tag: Todo documento HTML possui marcadores (do inglês: tags), palavras entre parênteses angulares (chevron) (< e >); esses marcadores são os comandos de formatação da linguagem.
